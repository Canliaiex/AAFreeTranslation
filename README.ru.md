# AAFreeTranslation

Плагин перевода чата для ArcheAge (поддерживает китайский, английский и русский языки). Требуется запуск PowerShell-сервиса перевода.

## Использование

### 1. Запуск сервиса перевода

Дважды щёлкните `run.bat`, чтобы запустить PowerShell-сервис перевода.

> При сворачивании окно скрывается в системный трей. Дважды щёлкните значок в трее, чтобы снова его отобразить.
>
> Примечание: если вам нужно отправлять сообщения через `Shift+Enter`, запускайте `run.bat` **от имени администратора** (правой кнопкой мыши → Запуск от имени администратора).

### 2. Установка плагина

Поместите папку `AAFreeTranslation` в каталог `Documents\AAFreeTo\Addon\`, затем добавьте строку `AAFreeTranslation` в файл `addons.txt`.

### 3. Настройка в игре

Откройте меню настроек в правом нижнем углу → **ADDON**, нажмите кнопку обновления в правом нижнем углу окна ADDON, чтобы загрузить плагин, выберите этот плагин из списка, затем нажмите кнопку Настройки справа для конфигурации.

Не редактируйте файлы вручную. Используйте окно настроек, описанное выше, для конфигурации плагина:

| Параметр | Описание |
|----------|----------|
| Автоперевод | Автоматически переводить входящие сообщения чата |
| Перевод ввода | Показывать окно ручного перевода |
| Движок перевода | Google Переводчик или ИИ-перевод |
| URL API / Ключ / Модель | Требуется для режима ИИ-перевода (по умолчанию GLM-4-Flash) |

> В режиме ИИ-перевода входная и выходная модель настраиваются отдельно. Если нет особых требований, укажите одну и ту же модель в обоих полях.

## Описание файлов

| Файл | Назначение |
|------|------------|
| `run.bat` | Запуск сервиса перевода (двойной щелчок) |
| `main.lua` | Игровой плагин |
| `settings_page.lua` | Панель настроек в игре |
| `monitor_translation_v2.ps1` | PowerShell-сервис перевода |
| `config.ini` | Файл конфигурации (настраивается в игре, не редактировать вручную) |
| `ai_prompts.json` | Промпты для ИИ-перевода |
| `cache/` | Директория кэша для обмена данными между плагином и сервисом |

## Известные проблемы

- Если в интерфейсе игры есть активный элемент ввода текста, результат перевода может быть вставлен не в поле чата, а в другой редактор.

## Примечания

- Проверенные модели: `glm-4-flash`, `deepseek-v4-flash`
- При использовании других моделей с функцией Reasoning/Thinking необходимо добавить код отключения мышления в функцию `Invoke-ChatAPI` в файле `monitor_translation_v2.ps1` (около строки 508). Пример:

```powershell
# Отключение мышления для моделей deepseek
if ($model -like '*deepseek*') { $body.thinking = @{ type = "disabled" } }
```
